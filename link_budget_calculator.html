Satellite Link Budget Calculator

What is a Link Budget?

A link budget is an accounting of all gains and losses in a radio communication link — from the transmitter, through the propagation medium, to the receiver. The goal is to determine whether there is sufficient signal power at the receiver to maintain the desired quality of service.

How to Use This Tool

Geometry tab — set satellite altitude and your elevation angle. The slant range is calculated automatically.

Uplink tab — model the ground→satellite link (your transmitter, the path, and the satellite receiver).

Downlink tab — model the satellite→ground link (satellite transmitter, path, and your receiver).

Summary tab — see both budgets side by side.

Color Coding

Yellow cells are user inputs.

Blue cells are calculated automatically.

Link margin is green (>3 dB), yellow (0–3 dB), or red (<0 dB).

Key Formulas
Slant Range
d = √[(Rₑ·sin θ)² + h(2Rₑ+h)] − Rₑ·sin θ Rₑ = 6378 km, h = altitude, θ = elevation angle
Free Space Path Loss
FSPL (dB) = 20·log₁₀(d_km) + 20·log₁₀(f_MHz) + 32.44
EIRP
EIRP (dBW) = P_tx (dBW) − L_feeder (dB) + G_ant (dBi)
Parabolic Antenna Gain
G (dBi) = 10·log₁₀(η) + 20·log₁₀(D_m) + 20·log₁₀(f_MHz) − 39.6 η = efficiency (typically 0.55–0.65)
System Noise Temperature
T_sys = T_ant + T₀·(L−1) + T_LNA·L L = feeder loss (linear), T₀ = 290 K
G/T Figure of Merit
G/T (dB/K) = (Ga − L_feeder − L_pol) − 10·log₁₀(T_sys)
S/N₀
S/N₀ (dBHz) = P_iso + G/T + 228.6 228.6 = −10·log₁₀(k_B), Boltzmann constant
Eb/N₀ and Link Margin
Eb/N₀ (dB) = S/N₀ − 10·log₁₀(bit_rate) Link Margin = Eb/N₀_rx − Eb/N₀_req − L_impl
